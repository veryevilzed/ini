# ini

## Installation

```
New instalation:
pip install -e git+https://github.com/veryevilzed/ini.git#egg=ini

or

pip install -e git+git://github.com/veryevilzed/ini.git#egg=ini


Upgrade:
pip install -U -e git+https://github.com/veryevilzed/ini.git#egg=ini

or

pip install -U -e git+git://github.com/veryevilzed/ini.git#egg=ini

```



## Request

```
ini -f ./config.ini -d ./default.ini section value
```
